# Vue Cli Template

> vue-cli 4 專案模版

方便快速建置專案及常用設定

## Quick usage

```script
vue create --preset BJ0815/vue-cli-template {{ PROJECT_NAME }}
```

## About

基本設定

- vue router (history mode)
- pre-render
- axios (包裝過一層客製攔截器)
- eslint (standard)
- git pre-commit lint

Options Plugins

- lodash
- vue-echarts
